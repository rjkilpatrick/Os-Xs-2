# Os-Xs-2
Noughts and crosses squared

## Usage

The usual CMake instructions:

1. `git clone --recurse-submodules https://github.com/rjkilpatrick/Os-Xs-2`
2. `mkdir build && cd build`
3. `cmake ..`
4. `cmake --build .`
5. `./os-xs-2`
